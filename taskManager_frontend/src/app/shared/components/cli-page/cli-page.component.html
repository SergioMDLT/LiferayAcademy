Initializr

CiviBlade is a Blade CLI extension developed by the creators of this website to automate the generation of Liferay workspaces with predefined configurations and embedded resources. It offers new users and developers in training a much easier entry into the Liferay ecosystem by allowing them to bypass initial setup steps and directly create portlets, services, and more.

You can explore the CiviBlade project repository and contribute your own custom commands at GitHub.

If this Initializr does not work, please download or compile civiblade.jar and execute it. Then, from the same location in a terminal, run: .\civiblade init
